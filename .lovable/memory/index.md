- Workflow: Audit existing -> Confirm -> Build surgically -> Clean dead code -> Post-audit. (See ~user).

- pgvector RPC: SECURITY DEFINER avec `search_path = public, extensions` obligatoire (sinon opérateur `<=>` introuvable).
