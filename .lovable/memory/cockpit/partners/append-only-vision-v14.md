# Memory: cockpit/partners/append-only-vision-v14
Updated: now

Vision de l'Espace Partenaire (v14.0) : "Pilotage en ajout seul" (Append-Only Piloting). Les partenaires peuvent lire les entités assignées (Missions, Leads, Documents, Solutions, RDV, Transcriptions, Annonces) et AJOUTER des données sans altérer les données CRM existantes gérées par l'admin (nlq@nlq.fr). Les partenaires sont rendus autonomes mais leurs activités ne doivent pas être "transformatrices" pour le Cockpit central. Les intégrations API Slack et Notion sont définitivement exclues. Les synchronisations d'agendas tiers ne sont pas une priorité mais ne sont plus définitivement exclues.
